Text formatting tags in HTML

  1. <b>(bold text) : This is a physical tag, which is used to bold the text written between it.
  2. <strong>(Strong tag) : This is a logical tag, which tells the browser that the text is important.
  3. <i>(italic tag) : This is a physical tag which is used to make text italic.
  4. <em>(Emphasised tag) : This is a logical tag which is used to display content in italic.
  5. <mark>(Mark tag) : This tag is used to highlight text.
  6. <u>(Underline tag) : This tag is used to underline text written between it.
  7. <sup>(Superscript tag) : It displays the content slightly above the normal line.
  8. <sub>(Subscript tag)It displays the content slightly below the normal line.
  9. <strike>(Strike tag) : This tag is used to draw a strikethrough on a section of text. (Not supported in HTML5)
